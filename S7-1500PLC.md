#### 局部報警案例

![image-20220617101309842](https://raw.githubusercontent.com/mengziup/pic/main/image-20220617101309842.png)

```scl
IF switch THEN
    autoLight := 1;
ELSE 
    autoLight := 0;
END_IF;

IF falt1 OR falt2 OR falt3 THEN
    alert := TRUE;
END_IF;

IF reset THEN 
    alert := FALSE;
END_IF;

IF alert AND clock_10Hz THEN
    faltLight := 1;
    faltSound := 1;
ELSE
    faltLight := 0;
    faltSound := 0;
END_IF;
```
